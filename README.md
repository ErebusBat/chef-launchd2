# launchd2 Cookbook

This is a fork of the official `launchd` Chef provider that differs in the following ways:

- Provides error checking for `start_calendar_interval` key
  - Submitted upstream: chef/chef#5115

### Platforms

- OS X

### Chef

- Chef 12.0 or later
